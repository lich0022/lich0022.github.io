HTML Tags Learned

HTML Anchors
By adding links using the Anchor tag, it allows us to be directed to other pages of our website or be directed to different websites.
HTML Images
Using the HTML Images tag, we are able to add any image of any size to our webpages.
HTML Lists
HTML Lists, such as this one, all us to create description lists, unordered and ordered lists and different ways to list information.
HTML Forms
Forms allow viewers to interact with your website.
HTML Tables
The HTML Tables tag are a way for you to organize information on your website as well as make schedules or timetables.
