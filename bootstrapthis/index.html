What is Bootstrap?

Bootstrap is a front-end open source toolkit which is very popular with website developers. It is a very easy system to use. Classes are added within the HTML in order to style it, instead of created a seperate CSS stylesheet.

Grids

Operating Grids

Grids in Bootstrap are an easy alternative to using grids in regular HTML. When using grids, the user must choose to use containers, rows, and columns. These align the content within the grids using certain breakpoints. There are six of these breakpoints.

The Breakpoints

Breakpoints are customizable widths which play a crucial part in the responsivenss of the user's site. The six breakpoints are: Extra small (xs), Small (sm), Medium (md), Large (lg), Extra large (xl), and Extra extra large (xxl). Each of these breakpoints have their own unique containers, class prefixes and modifiers.

Colours

Theme Colours

Bootstrap uses a subset of all colours for the purpose of having a smaller, simpler colour palette when users creat their colour schemes. These theme colours are Primary(Blue), Danger(Red), Secondary(Grey), Warning(Yellow), Success(Green), Info(Lighter Blue), and Dark(Black).

Buttons

Button Colours

In Bootstrap it is very easy to change the colour of the user's buttons. The process is very similar to changing text colour. The user would use "btn-" and then the name of the colour they'd want to use. For example, "btn-danger" would make the button red, and "btn-success" would make the button green.
